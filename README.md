## Features

- **One-Click Download** - Automatic download and installation of older Roblox versions
- **Auto-Update** - Always stay up-to-date with the latest features
- **Multi-Instance** - Launch multiple Roblox instances simultaneously
- **Smart Version Manager** - Automatically manages and caches downloaded versions
- **Fast Downloads** - Optimized headless browser for faster downloads
- **Beautiful UI** - Modern, intuitive user interface

## What Does It Do?

TiRex Downgrader automatically downloads older versions of Roblox and allows you to run multiple instances at once. Perfect for:

- Running multiple accounts (future)
- Downgrading and using old version

## Installation

1. Download the latest **[TiRex-Downgrader.exe](https://github.com/dyr14n/tirex-downgrader/releases/latest)**
2. Run the installer
3. App will automatically install and open
4. That's it!

> **Note:** Windows Defender might flag the app. This is a false positive. Click "More info" ‚Üí "Run anyway"

## Usage

1. **Launch the app** from Desktop or Start Menu
2. **Click "Download & Launch"** (or wait for auto-start)
3. **Wait** for the download and extraction to complete
4. **Roblox launches automatically** with the selected version
5. **Click "Launch More"** to open additional instances

### Auto Download & Extract

## Troubleshooting

<details>
<summary><b>App won't open?</b></summary>

- Right-click ‚Üí **Run as Administrator**
- Check if **antivirus** is blocking it
- Disable **Windows Defender SmartScreen** temporarily
- **Reinstall** the app

</details>

<details>
<summary><b>Download failed?</b></summary>

- Check your **internet connection**
- Make sure you're using the **latest version**
- Try **restarting** the app
- Clear the **downloads folder** in AppData
- Check if **GitHub** is accessible

</details>

<details>
<summary><b>Roblox won't launch?</b></summary>

- Make sure the **download completed** successfully
- Check if **RobloxPlayerBeta.exe** exists in versions folder
- Try **deleting** the version folder and re-downloading
- Run app as **Administrator**

</details>

<details>
<summary><b>Update not working?</b></summary>

- **Close and reopen** the app
- Check if **GitHub** is accessible
- **Manual download** latest version
- Check your **firewall** settings

</details>

<details>
<summary><b>Multiple instances not working?</b></summary>

- Make sure the first instance **launched successfully**
- Wait for the download to **complete** before launching more
- Check if you have enough **RAM**
- Close unnecessary programs

</details>
